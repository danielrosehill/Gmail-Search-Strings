# Just a few Gmail searches!

 A few search strings which can be useful for searching through an inbox in order to find items matching a specific keyword. Search org for creating filters in Gmail, Google Workspace, Mail, etc. 

 These can also be useful when creating automations using Zapier, as Zapier allows users to bake filters into the automations.
